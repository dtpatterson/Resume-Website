NobleBookBarn

Online Bookstore Crafted From Surveys

  • GitHub - Project Source Code
  • NobleBookBarn Documentation

Collaborated with a small team to develop a full-stack website for a User Interfaces class at UTSA.

Utilized real data collected from curated surveys to construct an online bookstore, catering to the preferences of the participants.

The purpose of the website is to create a tailored platform that aligns with the preferences and needs of our target users, as identified through our survey findings.

Technologies included:

  • Front-end: Next.js
  • Back-end: Java Spring Boot
  • DataBase: MySQL
  • Collaboration: GitHub, Discord
