National Park Chooser Application - Developed JavaFX Application for School Project

  • GitHub - Project Source Code

Worked with a small team of 4 to develop a web application for a Applications Programming class at UTSA.

The purpose of the application is to provide a convenient and easy-to-use tool for people who are interested in visiting national parks. The application consisted of all 63 parks, including pictures and descriptions for each.

Technologies included:

  • Java - using SOLID principles and MVC design pattern.
  • JavaFX - “GUI, graphics, and multimedia API of the future”
  • SceneBuilder - graphical GUI builder used in conjunction with JavaFX
