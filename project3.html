BonVoyage

National Park Chooser Application

  • GitHub - Project Source Code

Collaborated with a dynamic team of four to craft a user-friendly web application as part of an engaging Applications Programming class at UTSA.

Our mission? To deliver a seamless and enjoyable experience for travel enthusiasts exploring the beauty of national parks. The application showcased all 63 parks, complete with captivating pictures and detailed descriptions.

Technologies included:

  • Java - Implementing SOLID principles and the elegant MVC design pattern.
  • JavaFX - Unleashing the power of the "GUI, graphics, and multimedia API of the future."
  • SceneBuilder - A sleek graphical GUI builder seamlessly integrated with JavaFX.
